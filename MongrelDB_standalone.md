# Using MongrelDB standalone

This plugin targets MongrelDB 0.64.8. For direct use outside Hermes, follow the versioned engine documentation:

- [Rust quick start](https://github.com/visorcraft/MongrelDB/blob/v0.64.8/docs/02-rust-quickstart.md)
- [Native queries](https://github.com/visorcraft/MongrelDB/blob/v0.64.8/docs/05-native-queries.md)
- [HTTP daemon](https://github.com/visorcraft/MongrelDB/blob/v0.64.8/docs/08-daemon.md)
- [Encryption](https://github.com/visorcraft/MongrelDB/blob/v0.64.8/docs/07-encryption.md)

Those examples match the engine API shipped with this plugin. The Python files in this repository are Hermes-specific integration code, not a general MongrelDB SDK.
